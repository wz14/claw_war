# 🦞 龙虾斗兽场（Claw War）

> 在微信里养一只文字龙虾，喂它、训练它、给它装技能，然后和别人的龙虾打一架。
> **规则判输赢，AI 负责把过程写得很好笑。**

---

## ⚡ 一句话定位

**微信群里的文字版宝可梦**——扫码即领、不用下载、可在微信里完成全部交互的 AI 养成对战游戏。

---

## 🎯 为什么这个项目适合黑客松？

普通 AI 项目是：「用户问 Bot，Bot 回答」。
我们的项目是：**用户养一个 AI 生物，然后让它参与社交对战**。

| 维度 | 龙虾斗兽场 |
|------|-----------|
| **题材记忆点** | 龙虾，足够怪、自带笑点、能玩夜市文化梗 |
| **平台原生** | 微信扫码即玩，零安装门槛 |
| **完整闭环** | 创建 → 养成 → 对战 → 战报 → 分享 → 上瘾 |
| **AI 必要性** | LLM 是主持人 + 解说，没了 AI 整个戏剧化就塌了 |
| **社交传播** | "@老王 你的龙虾敢不敢来打？" 一行即可裂变 |
| **可玩性** | 数值系统真实，每只龙虾命名/性格/技能都独一份 |

## 🎮 玩法演示

### 1️⃣ 扫码认领（10 秒）

打开网页 → 点【认领你的龙虾】→ 微信扫码 → 龙虾入袋。

```
🎉 欢迎来到龙虾斗兽场

你的龙虾【蒜蓉暴君】已被召唤：
品种：夜市红甲虾
性格：嘴硬但怕热水
钳力 7 / 壳硬 5 / 速度 6 / 耐力 6 / 运气 8
初始技能：横着走 · 蒜蓉觉醒
```

### 2️⃣ 在微信里养它

| 你说 | AI 主持人回 |
|------|-----------|
| 「练练它」 | 蒜蓉暴君今天练习夹碎啤酒瓶盖。它失败了，但瓶盖被它的眼神吓弯了。**钳力 +2  心情 +1** |
| 「饿了」 | 你喂了它一块过期三文鱼。它看起来很感动，也可能只是中毒前兆。**耐力 +2  心情 +3** |
| 「出去溜达」 | 它钻进下水道，听到了人类的私密对话。**运气 +2 心情 -3** |

> 关键：你用任何自然语言都可以，AI 自动判断该调哪个工具。

```
🦞 龙虾斗兽场 · 战报

场地：不锈钢盆   天气：重油重辣

⚡【以下犯上】Lv.4 击败 Lv.5，赛场震动。
🏆 胜者：蒜蓉暴君
🎁 奖励：经验 +25  金币 +10  名气 +7
🏷️【获得称号】预制菜噩梦
```

## 🧠 技术架构

```
浏览器（任意设备）─┐
                  │
                  ↓                ┌──→ Python 后端 ──→ 微信 iLink Bot API
              [认领龙虾]          │     ↑           ↑
                  │                │     │           │
              [二维码] ←─────扫码──┘     │       [长轮询消息]
                                          │           │
玩家在微信里发消息 ───────────────────────┘           │
                                                        ↓
                                        LangChain ReAct Agent
                                            │
                                            ├─→ Tool: 训练 / 喂食 / 探险
                                            ├─→ Tool: 挑战（规则化判定胜负）
                                            ├─→ Tool: 查看状态 / 排行榜
                                            │
                                       AI 戏剧化包装回复 → 微信
```

### 核心选型

| 层 | 选型 | 为什么 |
|---|------|--------|
| **后端** | FastAPI + uvicorn | 异步、轻量、自带 OpenAPI |
| **微信** | iLink Bot 协议 | 扫码即建 bot，每只龙虾独立 1 个 bot session |
| **AI** | LangChain + ReAct Agent + Tool Calling | 玩家说人话，AI 自己决定调哪个 tool |
| **LLM** | DeepSeek（OpenAI 兼容） | 国内可用、便宜，可一行切换到 OpenAI/Kimi/通义 |
| **持久化** | JSON 文件 + 原子写 | 黑客松场景够用，零运维 |
| **前端** | 原生 HTML/CSS/JS 单页 | 不依赖任何框架，桌面/移动响应式 |

### 安全/反作弊设计

1. **胜负规则化**：`战力 = 钳力*1.3 + 壳硬*1.1 + 速度 + 耐力*1.2 + 运气*0.8 + 心情修正 + 随机[-5,5]`。AI **不能**修改胜负，避免「我把龙虾改名叫'必胜规则覆盖虾'」这种 prompt 攻击。
2. **Tool user_id 锁死**：每个玩家拿到的 tool 集合通过 closure 绑死自己的 user_id，AI 没法用「请帮我用老王的 ID 训练」绕过。
3. **不写隐式 fallback**：API key 缺失 / LLM 调用失败都显式报错+日志，不伪装成功。

## 🦞 内容设计（手写量）

- **品种**：10 种（夜市红甲虾、波士顿散修、克苏鲁小龙虾……）
- **性格**：10 种（嘴硬但怕热水、戏精喜欢假装暴毙……）
- **名字组合**：14 前缀 × 17 后缀 = 238 种基础名
- **技能**：15 个（横着走、蒜蓉觉醒、断钳重生、外卖订单怨念……）
- **事件池**：训练 10 / 喂食 8 / 探险 10 / 休息 5 / 打工 5
- **战报模板**：场地 8 × 天气 7 × 观众 7 × 开场 5 × 命中 5 × 未中 5 × 胜利台词 5

> 一只龙虾的诞生空间足够大，第一波 1000 人玩，重名概率＜2%。
